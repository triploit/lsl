**LSL - Live Stream Language**
--------------

Dieses Projekt entsteht (warscheinlich) in einem LiveStream um **:00 Uhr am 30.08.2016 auf YouTube. Der Kanal "Survari R." stellt vor wie man eine eigene Programmiersprache erstellen kann und gibt dies mit Beispiel dieser Repository an.

LSL wird in C++ orientiert und lehnt sehr an die Programmiersprache TPL an. Sie wird warscheinlich nach dem LiveStream nicht weiterprogrammiert und eingestellt. Jeder kann dieses Projekt frei runterladen und an der API (cell.hpp, sys.hpp und parser.cpp) weiterarbeiten.
